# To-Do List Application in Python

A simple command-line based To-Do List application built using Python. This project helps users to add, view, update, and delete daily tasks efficiently.

Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Simple menu-driven interface

Technologies Used
- Python

How to Run the Project
1. Clone the repository:
   git clone https://github.com/your-username/to-do-list-python

2. Navigate to the project folder:
   cd to-do-list-python

3. Run the program:
   python todo.py
